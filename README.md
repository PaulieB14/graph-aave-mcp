# graph-aave-mcp

MCP server for querying AAVE V2/V3 lending protocol and governance data via The Graph.

## Setup

Set your Graph API key:
```
GRAPH_API_KEY=your_key_here
```

Get a free API key at https://thegraph.com/studio/apikeys/

## Supported Chains

- Ethereum V3 & V2
- Base V3
- Arbitrum V3
- Polygon V3 & V2
- Optimism V3
- Avalanche V3 & V2
- Fantom V3
- AAVE Governance V3
